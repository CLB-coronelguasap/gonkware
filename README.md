# GonkWare

## Requirements

- Python 3.8+
- pygame 2.x

## Setup

1. Install dependencies:
   ```sh
   pip install pygame
   ```

2. Project structure:
   ```
   gonkware/
   ├── main.py
   ├── assets/
   │   ├── bg_blur.jpg
   │   ├── audio/
   │   │   └── title.opus
   │   ├── font/
   │   │   └── PhillySans.ttf
   │   └── icons/
   │       ├── play.png
   │       ├── settings.png
   │       └── quit.png
   └── README.md
   ```

3. Run the project:
   ```sh
   python main.py
   ```

## Controls

- Arrow keys: Navigate menu items and settings
- Enter/Space: Select menu item or toggle setting
- Mouse: Click and drag sliders, click buttons
